⚠️ 已發現的架構問題
🔐
硬編碼密碼 / 明文 Secrets
MySQL 密碼 (springboot123)、JWT secret 直接寫在設定檔與 K8s YAML 中，未使用 Kubernetes Secret
📦
SQLite 聊天 DB 與 K8s 不相容
聊天訊息存在本地 chat.db 檔案，多副本或 Pod 重啟後資料會消失，應遷移至 MySQL 或 Redis
🔀
Kong 路由無持久狀態
Kong 路由透過腳本動態設定，K8s Pod 重啟後需重新執行配置
📈
HPA 最大副本數設為 1
無法自動水平擴展，失去 HPA 意義
🤖
AI 功能未完整整合
AiController 存在但被停用，Ollama 部署有對應 YAML，但尚未與主應用連通
💾
JPA DDL: update (非 validate)
生產環境應使用 validate 或 Flyway/Liquibase 管理 schema，避免意外自動修表
